Данное консольное приложение выполняет функцию калькулятора.

Созданный калькулятор имеет базовые арифметические функции:
сложение;

вычитание;

умножение;

деление.


Также в калькуляторе предусмотрены расширенные функции:

расчет процента от числа;

возведение числа в степень;

логарифм числа.


Для запуска кода, нужно скопировать код и вставить его в свой проект, далее запускить код.
